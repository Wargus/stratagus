Invasion - Battle Of Survival
-----------------------------

About
-----
Invasion - Battle of Survival is a real-time strategy game using the
Stratagus game engine.

How to install
-------------

Unpack the game data in a games directory:
> tar -xzvf Invasion-bos.tgz

You also need to get version 2.2 of the Stratagus engine available at
http://www.stratagus.org.


How to start the game
---------------------
Execute the folowing command in the stratagus directory:
> ./stratagus -d path-to-games-directory/data.bos
